# Project 00 — Raspberry Pi Setup and Git Workflow

## Overview
This project establishes the foundation for all subsequent Cyber-Physical Systems development. It focuses on setting up a Raspberry Pi environment, enabling remote access, and integrating version control using Git and GitHub.

---

## Objective
- Connect to a Raspberry Pi via SSH
- Configure wireless networking
- Clone and manage a remote Git repository
- Establish a reproducible development workflow for embedded systems

---

## Implementation

### Remote Access (SSH)
The Raspberry Pi was accessed using SSH, allowing command-line interaction over the network. This replaces traditional UART-based serial communication and enables scalable development.

### Git & GitHub Integration
- Created and cloned a shared repository
- Practiced committing, pushing, and pulling changes
- Established a structured workflow for collaborative embedded development

---

## Skills Developed
- Linux command-line usage
- SSH-based remote development
- Git version control fundamentals
- Repository management

---

## Tools & Technologies
- Raspberry Pi 3B
- Linux (Raspberry Pi OS)
- Git / GitHub
